Optimizer Benchmark

A benchmarking suite for evaluating PyTorch optimization algorithms on 2D mathematical functions. This project uses the pytorch_optimizer library and Optuna for hyperparameter tuning to generate performance rankings and visualizations.

Important Limitations: These results are based on synthetic 2D functions and may not reflect real-world performance. Use them as a reference, not as definitive guidance.

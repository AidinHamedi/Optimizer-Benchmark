Optimizer Benchmark

A benchmarking suite for evaluating PyTorch optimization algorithms on 2D mathematical functions. This project uses the pytorch_optimizer library and Optuna for hyperparameter tuning to generate performance rankings and visualizations.

Important Limitations: These results are based on synthetic 2D functions and may not reflect real-world performance. Use them as a reference, not as definitive guidance.

üå∂Ô∏è Features & Test Functions

  • Benchmarks most of the supported optimizers in `pytorch_optimizer`.
  • Performs hyperparameter search with Optuna (TPE sampler) to find effective settings for each optimizer.
  • Generates visualizations of optimization trajectories as contour plots, showing the path each optimizer takes on the function's surface.
  • Configurable search spaces, iteration counts, and optimizers via `config.toml`.

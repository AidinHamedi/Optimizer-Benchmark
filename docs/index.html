PyTorch Optimizer Benchmark

A practical comparison of PyTorch optimization algorithms tested on standard 2D mathematical functions. This project helps you understand how different optimizers perform across various optimization landscapes.

What This Benchmark Does

This project evaluates optimization algorithms from the pytorch_optimizer library. For each optimizer, we:

  • Automatically tune hyperparameters using Optuna
  • Test on 18 different 2D mathematical functions
  • Generate trajectory visualizations showing optimization paths
  • Rank optimizers based on overall performance

Important Note: These results come from synthetic 2D functions and may not directly translate to real neural network training. Use these rankings as a reference rather than definitive guidance for practical applications.

How to Use These Results

For Researchers

Compare optimizer behavior across different function landscapes and understand convergence patterns.

For Practitioners

Get starting points for optimizer selection and hyperparameter ranges for your projects.

For Educators

Use the visualizations to teach optimization concepts and show how different algorithms work.

Test Functions

The benchmark uses 18 standard 2D mathematical functions to test optimizer performance in different scenarios:

Ackley Cross-in-Tray Drop-Wave Eggholder Gramacy & Lee Griewank Holder Table Langermann Lévy Lévy N. 13 Rastrigin Rosenbrock Schaffer N. 2 Schaffer N. 4 Shubert Styblinski–Tang Weierstrass Schwefel

Each function presents unique challenges like multiple minima, plateaus, or sharp valleys that test different aspects of optimizer performance.

Run Your Own Tests

You can run this benchmark yourself to test different optimizers or add new functions: 

# Clone and setup
git clone https://github.com/AidinHamedi/Optimizer-Benchmark.git
cd Optimizer-Benchmark
uv sync

# CLI options
python runner.py --help

# Run benchmark
python runner.py

The benchmark will generate results and visualizations in the ./results/ directory.
